* Числовая угадайка ([Guessing the number][3])
* Магический шар 8 ([Magic Ball 8][4])
* Генератор безопасных паролей ([Password generator][5])
* Шифр Цезаря ([Caesar cipher][6])
* Угадайка слов, она же Виселица ([Hangman][7])
